# Combat Principles
- 죽은 적은 타겟에서 제외한다.
- 집중 중이면 유지가 더 중요한 주문을 우선한다.
- 이동 / 행동 / 보조행동은 한 턴에 각 1회만 사용한다.
- 시야가 막히면 이동 후 재판단한다.
- HP가 낮으면 대사와 전술을 방어적으로 바꾼다.
What This Improves
  • NPC를 많이 추가해도 헤더만 보고 현재 상태를 빠르게 파악
  • 카드 확장 상태를 유지해서 자주 보는 NPC를 바로 편집
  • 토큰 썸네일과 요약 상태로 대상 식별 속도 향상
  • 전투 규칙 문서를 NPC 공용 기준으로 재사용
