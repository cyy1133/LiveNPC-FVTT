# 커뮤니티 소개글 (KR)

## 1. Discord용 짧은 소개글

`FVTT AI NPC Runtime`이라는 Windows 데스크톱 앱을 만들었습니다.

Foundry VTT, Discord, LLM을 하나로 연결해서 NPC가 캐릭터답게 대화하고, 전투에서는 현재 상태를 보고 좀 더 말이 되는 턴을 수행하게 하는 런타임입니다.

핵심 기능:
- EXE 기반 실행
- Discord + Foundry 통합
- 죽은 적 / HP 0 / 비전투 참가 적 제외
- DND5e 행동경제 반영
- LOS / 벽 / 경로 / difficult terrain / cover 반영 1차 전술 판단

## 3. 긴 소개글

`FVTT AI NPC Runtime`이라는 Windows 데스크톱 런타임을 만들었습니다.

목표는 단순합니다. Discord 대화와 Foundry VTT 전투를 묶어서, GM이 NPC를 일일이 손으로 조작하는 부담을 줄이는 것입니다.

현재 되는 것:
- Discord, Foundry VTT, LLM을 한 앱에서 연결
- NPC별 Markdown 문서 기반 말투/행동 규칙 적용
- 전투 상태를 읽고 턴 행동 결정
- 죽은 적, HP 0 적, 비전투 참가 적 제외
- DND5e 행동경제 반영
- LOS, 벽, 경로, difficult terrain, cover를 고려한 1차 전술 판단
- 이동 / 행동 / 보조행동 / 짧은 대사를 순서대로 실행하고 턴 종료

현재 릴리스 기준:
- Windows EXE 제공
- Discord / FVTT / LLM Quick Setup UI
- 토큰 썸네일과 접힘 카드 기반 NPC 패널
